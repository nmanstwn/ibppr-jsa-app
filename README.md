# ibppr-jsa-app
Aplikasi berbasis web untuk membuat dan mengelola formulir Identifikasi Bahaya, Penilaian dan Pengendalian Risiko (IBPPR) serta Job Safety Analysis (JSA) secara digital. Data JSA otomatis tersimpan dalam JSA Log dan dapat langsung dihasilkan menjadi dokumen PDF untuk memudahkan administrasi, dokumentasi, dan pengendalian keselamatan kerja.
